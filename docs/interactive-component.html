6.2 Descriptions

The interactive part of our analysis was built with D3 version 6, which the code has been uploaded to vizhub. Due to time limit, we only create the interactive component for air quality dataset in NYC.The date we use are from NYC open data as described in previous sections, but formats of certain columns were changed to adapt to the plot. You can find the data here under the data folder of our repo.

6.3 Instructions for using the map

The plot is initialized with a histogram of mean PM 2.5 values in different boroughs.

  1. Users can press the bottons below the graph, you can see the mean NO2 and O3 values for different boroughs.

  2. Users can hover the mouse over a bar to see the difference between values for this borough and other boroughs.
